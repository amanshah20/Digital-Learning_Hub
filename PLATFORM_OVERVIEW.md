## 🚀 Quick Start (3 Steps!)

### Step 1: Install Dependencies (2 minutes)
```bash
cd g:\Semester-7\Capstone_Project
npm install
cd server && npm install
cd ../client && npm install
```

### Step 2: Setup Database (30 seconds)
```bash
cd server
npm run db:migrate
```
Creates admin user: **admin@eduverse.com** / **Admin@123**

### Step 3: Start Everything (1 command)
```bash
cd ..
npm run dev
```

**Access:** http://localhost:5173
